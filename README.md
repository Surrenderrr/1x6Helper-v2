1x6Helper v2 это улучшенная версия 1x6Helper v1, написанная на более современном фреймворке Avalonia.
Программа разработана для игроков кастомной карты Dota 1x6.
Все данные программа получает с серверов официального сайта (https://dota1x6.com) разработчика Dota 1x6, которые находятся в открытом доступе.
Features: 
1. Просмотр союзников игрока в последних 20 матчах режима 2x8.
2. Просмотр рейтинга игрока в режимах 1x6 и 2x8 за прошедший сезон.
3. Просмотр общих матчей двух игроков.
4. Просмотр кастомного параметра - порядочность игрока.

Расчет порядочности игрока:
Выбирайте героев и их легендарные таланты, которые считаете дисбалансными по сравнению с другими.
Программа запрашивает последние 25 матчей игрока и проверяет сколько из этих матчей были сыграны на персонажах с легендарными талантами из ранее составлено вами списка.
Число таких матчей делится на 25, в результате получается процент порядочности игрока.
Чем выше процент порядочности игрока, тем чаще он играет на дисбалансных по вашему мнению персонажах.
